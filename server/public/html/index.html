Comenzando

Bienvenido al Anime API, un proyecto personal que tengo como trabajo para la materia de Programación por Componentes en la Universidad Distrital Francisco José de Caldas. El objetivo de esta API es practicar la creación de servicios y aprender nuevas herramientas, tecnologías y conceptos que mejoren mis habilidades como desarrollador.

Este proyecto busca implementar una API para gestionar información sobre animes, personajes y otros recursos relacionados, utilizando un conjunto de tecnologías modernas. Durante el desarrollo, se abordan conceptos fundamentales de programación como la arquitectura MVC, autenticación, CORS, y más.

Te invito a explorar la API y la documentación completa para aprender más sobre su uso.

Tecnologías Utilizadas

La Anime API ha sido desarrollada utilizando un conjunto de tecnologías modernas que me han permitido mejorar mis habilidades de programación. A continuación, se detallan algunas de las principales herramientas y conceptos utilizados en el proyecto:

  • Node.js - Entorno de ejecución para JavaScript que permite construir aplicaciones de alto rendimiento en el servidor.
  • TypeScript - Lenguaje de programación que mejora la escalabilidad y la seguridad del código al introducir tipado estático en JavaScript.
  • Prisma - ORM utilizado para interactuar con la base de datos SQLite de manera eficiente y segura.
  • Turso con SQLite - Base de datos ligera y robusta que almacena la información de animes y personajes.
  • CORS - Middleware que gestiona las solicitudes de diferentes dominios, permitiendo acceso controlado a los recursos de la API.
  • Autenticación - Implementación de autenticación basada en tokens Bearer para asegurar el acceso a la API.
  • Vitest y Jest - Herramientas de testing para garantizar la calidad y fiabilidad del código mediante pruebas unitarias.
  • MVC - Patrón arquitectónico utilizado para separar las preocupaciones y estructurar el código de manera modular.

Este conjunto de herramientas me ha permitido crear una API robusta y escalable, al mismo tiempo que profundizo en conceptos clave de desarrollo y arquitectura de software.
